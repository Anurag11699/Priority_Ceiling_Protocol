SIMULATION OF PRIORITY CEILING PROTOCOL IN A SINGLE CORE SYSTEM

How to compile: 
make or make all

How to execute:
./test to execute

Directory Structure:

text files
tasks.txt: Task Set input.
resources.txt: Resource Set input.
OUTPUT.txt: Program Output.

.h files
configuration.h: input and output file names
data_structures.h: contains all needed data data structures for the simulation.
functions.h: contains all function declarations.

.c files:
priority_queue_functions.c: contains all functions related to the priority queue.
scheduling_functions.c: contains all the functions related to the scheduler.
